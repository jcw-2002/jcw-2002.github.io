# 什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架（静态网站生成器）。它使用 Node.js 构建，可以将 Markdown 文件（或其他渲染引擎支持的格式）解析并生成漂亮的静态 HTML 网页。

Hexo 的主要优点：

  • 极速生成： 得益于 Node.js，Hexo 可以快速生成包含数百个文件的静态网站。
  • Markdown 支持： 你可以使用熟悉的 Markdown 语法来撰写文章，Hexo 会将其转换为网页。
  • 强大的扩展性： 拥有丰富的插件和主题生态系统，可以轻松扩展功能和定制外观。
  • 一键部署： 支持将生成的静态文件轻松部署到 GitHub Pages、Vercel、Netlify 等多种平台。

# 什么是 ShokaX 主题？

当你使用 Hexo 时，网站的外观和布局是由 主题（Theme） 控制的。Hexo 拥有大量由社区贡献的主题。

ShokaX 就是一款非常受欢迎的 Hexo 主题（它是基于 Shoka 主题的改进或分支版本）。它以其现代化设计、流畅的动画效果（如 Pjax 无刷新加载）、丰富的功能和高度可配置性而闻名。

选择 ShokaX 的理由可能包括：

  • 美观的界面： 提供优雅、简洁的视觉体验。
  • 用户体验： 支持 Pjax 等技术，提升页面切换速度和流畅度。
  • 功能集成： 通常内置了评论系统、代码高亮、图片懒加载、搜索等多种实用功能。
  • 自定义选项： 提供丰富的配置项，让你能方便地调整主题细节。

# 如何使用 ShokaX？

通常，使用一个 Hexo 主题的步骤如下：

  1. 下载主题： 将 ShokaX 主题文件下载到你的 Hexo 站点的 themes 目录下（通常使用 git clone ）。
  2. 配置主题： 修改 Hexo 站点的根配置文件 _config.yml ，将 theme 字段的值改为 shokax （或者你存放主题的文件夹名称）。
  3. 个性化配置： 修改 ShokaX 主题自身的配置文件（通常是 themes/shokax/_config.yml ），根据你的需求开启 / 关闭功能、修改样式等。

具体的安装和配置方法，请务必参考 ShokaX 主题的官方文档或 GitHub 仓库（因为 ShokaX 可能有不同的维护者和版本，请查找你使用的具体版本的说明）。
